# Rule-Based AI Chatbot
## Project Description
This project is a simple Rule-Based AI Chatbot developed using Python. The chatbot interacts with users and provides predefined responses based on user input using if-else conditions.

## Features
- Greets users
- Responds to common questions
- Provides help information
- Gives course-related information
- Exits when the user types "bye" or "exit"

## Technologies Used
- Python 3

## Files Included
- chatbot.py
- code_screenshot.png
- output_screenshot.png
- README.md

## How to Run
1. Install Python.
2. Save the program as chatbot.py.
3. Open a terminal in the project folder.
4. Run:
   python chatbot.py
5. Start chatting with the bot.

## Sample Output

You: hello
Bot: Hello! How can I help you?

You: what is your name
Bot: I am a Rule-Based AI Chatbot.

You: exit
Bot: Goodbye! Have a nice day.
